# Python仓库管理系统 python499

本项目是一个基于Python语言的仓库管理系统，采用 Django 框架进行后端开发，前端使用 Vue3 和 Ant-design-vue 技术栈。以下为项目的详细说明。

## 技术栈

- 后端：Python Django
- 前端：Vue3 + Ant-design-vue
- 开发工具：Pycharm
- 构建工具：
  - 后端：Pip
  - 前端：vite

## 运行环境

- 操作系统：Windows
- Python版本：3.8
- NodeJs版本：v16
- 数据库：mysql8.0

## 功能模块

系统实现以下主要功能模块：

- 商品管理
- 商品分类管理
- 用户管理

## 系统角色

本项目涉及以下角色：

- 管理员：负责管理商品、分类以及用户信息

## 目录结构

```
项目目录结构
|-- backend    # 后端代码目录
|   |-- app    # 应用代码目录
|   |-- static # 静态文件目录
|-- frontend   # 前端代码目录
|   |-- src    # 源代码目录
|   |-- public # 公共资源目录
|-- docs       # 项目文档目录
```

## 核心亮点

- 使用 Python Django，易于维护和扩展
- 前端采用 Vue3，性能更优，响应更快
- 使用 Ant-design-vue，界面美观、友好
- 提供详细运行教程，便于快速部署和启动

## 部署步骤

1. 安装 Python 3.8 和 NodeJs v16
2. 安装 Pycharm 和相关插件
3. 配置数据库 mysql8.0
4. 克隆代码到本地
5. 进入 backend 目录，使用 pip 安装依赖
6. 前端使用 vite 构建工程，并启动开发服务器
7. 访问前端项目地址，进行系统部署和运行

请注意，以上步骤为一般性描述，详细部署步骤请参考项目文档。
